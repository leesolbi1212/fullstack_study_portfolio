JPA (Java presistence API)
- 데이터베이스와 객체 (클래스)를 연결해주는 기술
- SQL 없이도 DB 작업을 할 수 있게 도와주는 ORM 기술
- 자바에서 표준으로 만든 인터페이스임.
    - node.js는 몽구스가 필요에 의해서 따로 만든건디..ㅋㅋ
- 대표적인 구현체 : Hibernate (SQL 매핑 기술) -> MyBatis 같은 것 (id로 불러서 씀)



지연로딩 (Lazy Loading)
- DB에서 데이터를 바로 가져오지 않고, 실제 사용할 때 가져오는 방식
- 만약 회원이 쓴 글 목록을 보고 싶다? (Member와 Posts는 연결되어있음)
    - 즉시 로딩 : Member를 불러오면 Posts도 즉시 같이 가져옴 (전체 멤버와 전체 post를 join해서 가져오고, member를 select 해서 회원별 글을 볼 수 있음)
    - 지연 로딩 : Member를 불러올 때는 Posts는 안 가져오고, 사용할 때 가져옴 (멤버 중 어떤 멤버에 대해서만 select 해서 post를 join하고 가져옴)

스프링 시큐리티 (Spring Security)
- 자바 웹 애플리케이션에서 로그인, 로그아웃, 권한, 인증 같은 보안 기능을 쉽고 강력하게 처리할 수 있는 프레임워크 (localhost:8080에서 뜨는 화면을 쓸 일은 없음)

* 동작흐름
1. 사용자가 /login 요청을 보냄
2. 스프링 시큐리티가 요청을 가로챔
3. 사용자의 아이디/비밀번호 확인
4. 인증 성공 시 세션에 사용자 정보를 저장
5. 인증된 사용자만 특정 url에 접근 허용
6. 권한 없으면 접근 차단 (403)
